# My Rice

## Installation
### Install dotify
```
git clone https://github.com/encore-ecosystem/dotify.git
cd dotify
uv sync
source .venv/bin/activate
```

### Install MyRice
```
git clone https://github.com/meshushkevich/MyRice.git
cd MyRice
dotify apply
```
