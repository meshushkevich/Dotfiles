# Dotfiles

## Prerequirements
- networkmanager
- git

## Step 1: Clone repository
```sh
git clone https://github.com/meshushkevich/Dotfiles.git
cd Dotfiles
```

## Step 2: Run installer
```sh
install.sh
```
